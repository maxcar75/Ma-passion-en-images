procédure XAMPP



Ouvrir XAMPP 
Cliquer sur Start pour les modules Apache
Cliquer sur Start pour MySQL si nécessaire
Aller dans le dossierC:\xampp\htdocs\
Créer un dossier pour le projet (exemple : monsite)
Mettre le code du site dans ce dossier (index.html ou index.php)
Ouvrir un navigateur
Taper http://localhost/monsite
Vérifier que le site s’affiche
Ouvrir l’invite de commande
Taper ipconfig
Récupérer l’adresse IPv4
Sur un autre PC connecté au même réseau
Ouvrir un navigateur
Taper http://ADRESSE_IP/monsite
Vérifier que le site s’affiche
